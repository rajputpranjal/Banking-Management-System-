# Banking Management System SQL Project

Description:

Developed a database-driven Banking Management System using Oracle SQL and PL/SQL to manage customer records, bank accounts, deposits, withdrawals, and transaction history. The system automates core banking operations with strong data integrity, validation, and transaction control.

Key Features & Responsibilities:

Designed and implemented relational database tables for Customers, Accounts, and Transactions with primary and foreign key constraints.

Used Oracle Sequences to auto-generate unique IDs for customers, accounts, and transactions.
